The Eleven Truly Necessary Sublime Shortcuts (Mac)

September 24, 2015

When all else fails, impress people with your ability to completely forgo your mouse. SHIT.IS.AWESOME.

  1. Select a line of code

    2. Command + Shift + Right arrow or Command + L

  2. Move a line of code up or down

    3. Command + Control + Arrow Up/Down

  3. Indent/Unindent a line

    4. Command + ] and Command + [

  4. Add multiple cursors

    5. Command + click

  5. Select a line, word by word

    6. Option + Shift + Arrow right/left

  6. Split Screen

    7. Command + Option + 2

  7. Toggle between full screen and partial screen

    8. Control + Command + F

  8. Find & Replace

    9. Control + F and Control + Option + F

  9. Select all occurrences of current selection

    10. Control + Command + G

  10. Comment/Uncomment current line

    11. Command + Backslash

  11. Navigate through folders

    12. Control + 0 then Arrow
